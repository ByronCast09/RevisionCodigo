Métodos Vulnerables a Inyección SQL

Estas páginas contienen vulnerabilidades de inyección SQL deliberadas para fines educativos.

Método 1: Login Vulnerable

Esta página de login es vulnerable a inyección SQL porque concatena directamente la entrada del usuario en la consulta SQL.

Ejemplo de ataque: Ingresa ' OR '1'='1 como nombre de usuario y cualquier contraseña.

Ir al Login Vulnerable

Método 2: Búsqueda Vulnerable

Esta página de búsqueda es vulnerable a inyección SQL porque concatena directamente la entrada del usuario en la consulta SQL.

Ejemplo de ataque: Ingresa ' OR '1'='1 para ver todos los usuarios o ' UNION SELECT 1,2,3,4 -- para probar ataques de unión.

Ir a la Búsqueda Vulnerable
